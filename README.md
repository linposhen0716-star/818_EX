# 818 進銷存管理系統 (818_EX)

這是一個專為「818 早餐店」設計的簡易進銷存管理系統。透過 Python 自動計算銷售量，並在 Web 介面上提供直觀的盤點與進貨建議。

## 系統功能
1.  **自動計算銷售**：讀取 POS 匯出的 Excel，自動換算材料耗用量。
2.  **盤點進貨介面**：美觀的網頁介面，分類顯示各廠商產品，並根據銷量給予進貨建議。
3.  **Excel 導出**：一鍵產生 `.xlsx` 進貨清單，方便廠商對帳與列印。
4.  **本地儲存**：資料儲存於瀏覽器，下次開啟時會保留上次填寫的內容。

## 檔案說明
- `index.html`: 管理介面主網頁。
- `JavaScript/main.js`: 核心邏輯與廠商資料庫。
- `update_sales_strict.py`: **重要工具**。每週跑一次，用來更新 `main.js` 裡的銷售數據。
- `mapping_dict.json`: 關鍵設定檔。對應 POS 產品名稱與系統內的正式名稱。
- `import_data.py`: 當需要大批次更新廠商清單時使用。

## 使用步驟
1.  將 POS 匯出的 Excel 檔案放入資料夾。
2.  執行 `update_sales_strict.py` 更新銷售量。
3.  點擊 `index.html` 進行盤點。
4.  填寫庫存後，點擊「存檔」下載 Excel。
